# 🌐 3D Portfolio

A modern, interactive 3D portfolio website built with React, Three.js, and Framer Motion.

## ✨ Features

- 🎮 Interactive 3D models using Three.js & React Three Fiber
- 🎨 Smooth animations with Framer Motion
- 📱 Fully responsive design with Tailwind CSS
- 📬 Contact form powered by EmailJS
- ⚡ Lightning-fast build with Vite

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React 18 | UI Framework |
| Three.js + R3F | 3D Rendering |
| Framer Motion | Animations |
| Tailwind CSS | Styling |
| EmailJS | Contact Form |
| Vite | Build Tool |

## 🚀 Getting Started
```bash
git clone https://github.com/Takshil27/3d-portfolio.git
cd 3d-portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📦 Build for Production
```bash
npm run build
```
